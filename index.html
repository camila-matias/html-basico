HTML - Principais tags

Este documento conterá um estudo sobre as principais tags básicas do HTML, suas descrições e possíveis casos de uso.

É importante lembrar que as tags são elementos que são envoltos de sinais de menor e maior (tag de abertura) e sinais de menor, barra e sinal de maior (tag de fechamento). Há também tags que são denominadas auto-fechantes, de modo que estas não tem tags de abertura e fechamento, mas são uma tag só aberta com um sinal de menor e fechada com barra e sinal de maior.

1. Cabeçalho e estrutura do documento HTML

doctype
É uma tag introdutória, encontrada em todos os documentos HTML, cujo propósito é evitar que os navegadores renderizem a página que uma forma não padrão, os chamados "quirks mode". Dessa forma, com o doctype, o documento é renderizado conforme as especificações do HTML e CSS.
html
É a tag que engloba todo o conteúdo do documento HTML, seja este conteúdo visível ao usuário ou não. Tem como atributo importante o lang, que indica o idioma no qual o documento HTML em questão está escrito.
head
É a tag que engloba os metadados (informações gerais) do documento HTML.
meta
É a tag que define as informações de metadados que não possuem tags específicas para seu propósito, sejam elas a codificação de caracteres utilizada (charset), tamanho inicial do documento (viewport), entre outros.
link
É a tag que indica uma relação entre o documento e recursos externos. Amplamente utilizado para relacionar folhas de estilo (CSS externo) e fontes não padrão com o documento.
title
É a tag que indica qual o título que será exibido na aba do navegador.
style
É uma tag que engloba regras de estilização para o documento (CSS interno), podendo ser utilizada ou não.
body
É a tag que representa o conteúdo do documento HTML que será visível para o usuário. Só é possível ter uma tag body por documento.
main
É a tag que define o conteúdo principal do documento HTML. Não é recomendado que se tenha mais de uma tag main no documento e, esta tag não pode ser tag filha de tags do HTML semântico article, aside, footer, header ou nav.
script
É a tag utilizada para relacionar scripts executáveis ao documento ou incluí-los no próprio documento. Esta tag pode ser posicionada antes ou depois do body, a depender se o script necessita, na sua execução, que o documento HTML esteja totalmente carregado ou não.

2. HTML Semântico e divisores

HTML Semântico

No HTML Semântico, temos 5 principais tags: header, section, article, aside e footer.

header
Define uma seção introdutória, onde se tem, normalmente, links de navegação, ícone para a home page da aplicação que se está visualizando ou o título da página.
nav
Define uma seção de navegação da página, em geral associada ao header.
section
Define uma seção dentro do documento cujo conteúdo possui um tema específico, como, por exemplo, um conjunto de notícias.
article
É uma tag que, em geral, define um subtópico específico de uma tag section. Muito utilizada em páginas de notícias, em que uma notícia pode estar contida em uma tag article.
aside
É uma tag que delimita uma seção lateral, cujo conteúdo se relaciona tangencialmente ao conteúdo principal (de uma section, por exemplo).
footer
É a tag que define o rodapé da aplicação, onde se inserem links da página, opções de contato, termos de uso, políticas de privacidade, entre outras informações.

Divisores

Há também algumas tags de divisões de documento comumente utilizadas como div, span, hr e br.

div
É um divisor genérico e não semântico que define um bloco de código.
span
É uma tag utilizada quando se gostaria de realizar uma separação inline de elementos. Geralmente utilizada para separação de itens para estilização diferenciada.
hr
Representa uma linha horizontal desenhada no documento, utilizada para divisão de seções.
br
Representa uma quebra de linha. Geralmente utilizada para criar uma separação em seções ou parágrafos de um documento HTML, sem a utilização de estilização CSS.

3. Textos e Hyperlinks

h1, h2, h3, h4, h5 e h6
Tags de título e subtítulos. Utilizados em hierarquias, sendo h1 o maior título entre todos e h6, o menor. É recomendado que haja apenas uma tag h1 no documento HTML, contendo o título principal da página, e que demais subtítulos sejam tags h2 ou menores.
p
Tag de parágrafo, delimita um bloco de texto.
a
Tag de hyperlink, cria uma conexão para outra parte do documento ou outra página fora do documento HTML em questão. Possui como atributo principal o href, que contém informação de para onde o hyperlink irá levar. Pode possuir outros atributos como target ou download, a depender da função do hyperlink.
em
Visualmente, aplica um estilo em itálico no texto. Semânticamente, é utilizada para indicar uma ênfase no texto.
strong
Visualmente, aplica um estilo em negrito no texto. Semânticamente, é uma tag utilizada para indicar um texto de alta importância no documento.
ins
Visualmente, aplica um estilo sublinhado no texto. Semânticamente, indica um bloco de texto que foi inserido no documento.
del
Visualmente, aplica um estilo riscado no texto. Semânticamente, indica um bloco de texto que deveria ser desconsiderado.

4. Listas e Tabelas

Listas descritivas

Utilizadas para listar e descrever itens.

dl
Inicializa uma lista de descrições
dt
Indica o título do item a ser descrito
dd
Descreve o item indicado na tag dt

Listas ordenadas e não ordenadas

Utilizadas para listagem de itens de forma ordenada ou não.

  • A tag ul inicializa uma lista não ordenada
  • A tag li indica um item de uma lista, podendo ser utilizada em ambas listas (ordenadas e não ordenadas)
  1. A tag ol inicializa uma lista ordenada
  2. Da mesma forma que na lista não ordenada, a tag li indica um item de uma lista

Tabelas

Utilizadas para visualização de informações em formato tabular.

table
Tag que inicializa uma tabela
thead
Tag que indica o header da tabela, ou seja, sua primeira linha
tbody
Tag que engloba o conteúdo da tabela
tfooter
Tag que indica o footer da tabela, ou seja, sua última linha
tr
Inicializa uma linha da tabela.
th
Indica uma célula de importância dentro da linha e tabela
td
Indica uma célula genérica dentro da linha e tabela

5. Mídias

img
Tag utilizada para inserção de imagens no documento HTML. Possui 2 atributos importantes: src onde é indicada a fonte do arquivo de imagem e alt que representa um texto alternativo para a imagem, tanto para melhorar a acessibilidade, quanto para caso a imagem não carregue corretamente.
figure
Tag utilizada para inserção de imagens, trechos de código, citações, entre outros tipos de visualização. Normalmente, possui a tag figcaption como filha, cujo objetivo é inserir uma legenda para a figura representada.
svg
SVG (do inglês Scalable Vector Graphics) é uma linguagem de marcação de gráficos. No HTML, a tag svg pode tanto representar imagens que são facilmente alteradas em dimensão ou cor, quanto pode, juntamente com CSS, criar estilizações para a página HTML ou, até mesmo, animações.
picture
Uma tag comumente utilizada quando se quer otimizar a qualidade de uma imagem a depender do dispositivo no qual o usuário esteja a vendo. Possui como tags filhas a tag source com os atributos srcset e media para indicar a fonte das imagens para diferentes tamanhos de dispositivo e uma tag img como a imagem default caso todos as condições de media sejam falsas.
audio
Tag utilizada para inserção de audios no documento HTML. Assim como a tag img, possui o atributo src para que seja indicada a origem do audio a ser inserido. Esta tag possui também vários outros atributos que definem a forma como o audio é reproduzido (automaticamente ou não), se será em loop, se terão controles de áudio visíveis, entre outros.
video
Tag utilizada para a inserção de vídeos no documento HTML. Assim como as tags img e audio, possui o atributo src para que seja indicada a origem do vídeo a ser inserido. Esta tag também possui vários outros atributos que definem uma imagem prévia, se os controles estarão visíveis ou não, se o vídeo reproduzirá em loop, entre outros.
